# Excel-report-project

# This project analyzes sales data of Vrinda Store using Excel to gain business insights such as monthly sales, top customers, and product performance.

# PROJECT OVERVIEW:
Vrinda Store Sales Analysis is a data analysis project created using Microsoft Excel.
The goal of this project is to understand sales trends, customer behavior, and product performance
to help the business make data-driven decisions.

# Tools & Technologies Used

- Microsoft Excel
- Pivot Tables
- Charts & Graphs
- Data Cleaning Techniques
- Basic Business Analytics

# Dataset Information

- Dataset contains sales data of Vrinda Store
- Includes information such as:
  - Order ID
  - Customer Name
  - Gender
  - Age
  - Product Category
  - Order Amount
  - Order Date

# Dashboard & Visualizations

An interactive Excel dashboard was created using pivot charts and slicers
to visualize sales trends and customer insights.
